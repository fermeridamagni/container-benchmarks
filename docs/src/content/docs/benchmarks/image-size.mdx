## Image Storage Comparison

Both OrbStack Docker and Apple Containers are OCI-compliant, meaning they pull the same images. However, how they store these images internally on macOS can affect the total disk footprint.

### Compressed Image Size on Disk

The sizes reported below represent the actual space taken on the host filesystem by the respective container image layers.

### Analysis

Both runtimes are highly efficient. Apple Containers generally matches OrbStack Docker's footprint because both utilize native macOS filesystem features (like APFS clones and sparse files) to store layers optimally.
